OCaml bindings to the MuPDF library
===================================

OCaml bindings to use the [MuPDF](https://mupdf.com/) library in order to manipulate pdf files.

Those are far from complete but [requests are welcome](https://github.com/smimram/ocaml-mupdf/issues).

## Getting started

All the useful functions are in the `MuPDF` module. You can have a look at the [test](test/test.ml) or at the [online documentation](https://smimram.github.io/ocaml-mupdf/mupdf/Mupdf/MuPDF/) for more information.
